# Модуль 6. Системы управления конфигурацией
## 6.6. Практическое задание. Автоматизированное развертывание Nginx + Apache при помощи СУК Ansible

### Что нужно делать

Настройте автоматическое развертывание веб-серверов Nginx и Apache с помощью Ansible на одном или нескольких удаленных серверах. Для этого создайте роли для установки и настройки каждого сервера, а также запустите проверку работоспособности развернутых сервисов.
